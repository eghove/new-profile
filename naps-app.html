NaPS App

A collaborative effort with my colleagues Alan Wagner and Andrew Burns, the National Parks App (NaPS App) utilizes the NPS.gov API to provide the user information about a National Park matching their search criteria, including information about campgrounds. The NASA.gov API is used to provide the user with the most recent land satellite imagery of the National Park, and the openWeather API provides current weather information for a given National Park. The application also uses Google Firebase to authenticate users.
